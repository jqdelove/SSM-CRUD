# SSM-CRUD
基于Maven搭建的SSM框架

## 说明
基于BootStrap的员工信息后台管理系统，完整实现了员工信息的增删改查。
该项目为前后端分离项目，后端是基于Maven的SSM框架搭建的。
如果该项目对您有帮助，您可以点右上角 "Star" 支持一下 谢谢！

## 技术选型
### 后端技术
MyBatis-->ORM框架
MyBatisGenerator-->数据层代码生成
PageHelper-->MyBatis物理分页插件
Hibernator-Validator-->验证框架

### 前端技术
BootStrap-->前端框架

## 环境搭建
### 开发工具
IDEA-->开发IDE
Navicat-->数据库连接工具
Axure-->原型设计工具
MindMaster-->思维导图设计工具

### 开发环境
JDK-->1.8
Mysql-->8.0
